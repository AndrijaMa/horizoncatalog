# Snowflake Horizon Catalog



Run the code in the "setup snowflake.sql" script
Once the steps in the "setup snowflake.sql" are complete setup you local pyspark. 

Setup pyspark in local environment (Jupyter Notebook)
### Create the conda environment
conda create --name horizon_irc -c conda-forge findspark=2.0.1 jupyter=1.0.0 pyspark=3.5.0
### Activate the environment
conda activate horizon_irc
